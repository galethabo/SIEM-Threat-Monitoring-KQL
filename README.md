# Azure AD Threat Detection Using KQL ##

## Overview ##
This project demonstrates my ability to detect and investigate common cyber threats using Microsoft Sentinel and Kusto Query Language (KQL).

## Skills Demonstrated ##
- Brute force detection
- Password spray detection
- Suspicious IP investigation
- Impossible travel detection
- Risky sign-in monitoring

## Tools Used  ##
- Microsoft Sentinel
- Azure AD Sign-in Logs
- KQL (Kusto Query Language)

## Key Outcomes
- Detected brute force and password spray attacks using KQL
- Identified suspicious login patterns and attacker IP addresses
- Built investigation timelines for compromised accounts
- Simulated real SOC analyst detection and response workflows
